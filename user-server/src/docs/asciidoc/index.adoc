[[overview]]
= 개요

Baseball Community API 문서입니다.

이 문서는 다음과 같은 API를 포함합니다:

* **인증 API**: 소셜 로그인, 이메일 로그인, 토큰 갱신, 중복 확인
* **비밀번호 API**: 비밀번호 재설정 플로우 (인증코드 전송, 검증, 재설정)

[[overview-http-verbs]]

== HTTP 상태 코드

본 REST API에서 사용하는 HTTP 상태 코드는 가능한 한 표준 HTTP와 REST 규약을 따릅니다.

|===
| 상태 코드 | 용례

| `200 OK`
| 요청을 성공적으로 처리함

| `400 Bad Request`
| 잘못된 요청을 보낸 경우. 응답 본문에 오류에 대한 정보가 담겨있다.

| `401 Unauthorized`
| 인증 실패

| `403 Forbidden`
| 권한이 없는 요청

| `404 Not Found`
| 요청한 리소스가 존재하지 않음

| `429 Too Many Requests`
| 너무 많은 요청을 보낸 경우 (쿨다운)

| `500 Internal Server Error`
| 서버 내부 오류가 발생함
|===
